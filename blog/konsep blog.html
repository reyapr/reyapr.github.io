Sampai pada saat ini hampir semua orang memiliki smartphone, bahkan sebagian orang ada yang sulit jika harus lepas dari sebuah gadget dalam jangka waktu yang cukup lama. Hadirnya sebuah smartphone juga meningkatkan penggunaan sebuah internet disetiap kalangan masyarakat, sehingga banyak e-commerce bermunculan. Hal tersebut juga berdampak pada terbukanya peluang usaha yang baru, yang membuat masyarakat semakin mulai penasaran untuk mempelajari web ini. Semakin meningkatnya kesadaran akan sebuah internet, sebagian orang pasti sudah mengenal apa itu HTML(Hypertext Markup Language). HTML adalah sebuah bahasa yang digunakan untuk membangun sebuah halaman web, namun web yang sering digunakan oleh orang pada umumnya tidak sepenuhnya menggunakan HTML. HTML merupakan sebuah struktur dari sebuah website, namun presentasional style dan fungsionalitas dari sebuah website berjalan menggunakan CSS dan bahasa pemograman seperti Javascript. Adapun cara penulisan dalam membuat HTML adalah sebagai berikut: Head dan Meta Data (bagian kepala HTML dan meta data) Body Sections (bagian-bagian badan HTML) 2.CSS CSS merupakan bahasa untuk membantu tampilan pada sebuah web yang sebelumnya sudah dibuat oleh HTML lebih responsive. Tentunya CSS digunakan jika sudah membuat HTML terlebih dahulu, dengan mengatur sebuah tampilan elemen-elemen yang ada pada HTML. Untuk dapat mengubah layout HTML dengan CSS, HTML perlu menyertakan terlebih dahulu syntax untuk menghubungkan dengan CSS didalamnya. Hal tersebut terbagi menjadi 4 metode agar CSS dapat menggunakan layout pada HTML: 1. Embed dalam dokumen HTML 2. Link ke file CSS terpisah. Dengan cara menaruh meta data berikut di head HTML. 3. Inline CSS pada elemen/tag HTML 4. Import CSS dalam file CSS, misalnya di dalam mainfile.css PEMILIHAN ELEMEN Untuk mengubah suatu tampilan dari halaman web, maka dibutuhkan pemilihan elemen terlebih dahulu dalam HTML. Elemen yang sudah dipilih diterapkan dalam sebuah CSS agar dapat dimodifikasi menggunakan property CSS. setelah menerapkan selector dibutuhkan kurung keriting setelahnya, yang diisi oleh property. Dengan menggunakan git programmer dapat mengetahui perubahan source codenya, terutama jika mempunyai project dengan team. Namun jika ingin mengerjakan sebuah project bersama secara online menggunakan git, membutuhkan sebuah tempat penyimpanan secara online, salah satunya dengan Github. Github merupakan tempat para developer untuk berbagi code dan berkolaborasi dalam membuat sebuah project software. untuk berbagi/berkolaborasi dalam membuat sebuah project software diperlukan sebuah repository sebagai tempat penyimpanan file yang diubah kedalam git.
